Shoutcast Tools:
https://de.shoutcast-tools.com/

Installation

- Upload des "shoutcast_tools" Verzeichnisses innerhalb Ihrer PHP-Fusion Installation im Ordner "/infusions".
- Aktivieren Sie die Infusion im Adminbereich.
- Rufen Sie die Einstellungen  ber "Infusions Shoutcastst Tools" auf und tragen Ihre werte ein.
- Rufen Sie die Panel Verwaltung auf: "System Admin - Panels".

- Aktivieren und verschieben Sie das gew nschte Panel an einer Position Ihrer Wahl.

Achtung: Nehmen Sie keine  nderungen am Panel Namen / Content vor.
Steuern Sie den Panel Namen auschließlich über die Einstellungen: "Infusion - Shoutcastcast Tools"
